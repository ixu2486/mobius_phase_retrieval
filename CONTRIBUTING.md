# Contributing

Thanks for contributing to Möbius Phase Retrieval SDK.

## Welcome Contributions
- Issues (bug reports, clarification requests, docs gaps)
- Benchmark reports and reproducibility notes
- Public adapter examples around the application-layer retrieval boundary

## Scope Boundary
Please do not submit or request exposure of RetryIX private runtime internals, including:
- MemoryKernel internal scheduling details
- VirtualPIM resident execution paths
- TurboQuant-compatible backend internals
- SPD/EEPROM physical contract logic
- Hardware-specific routing and private execution paths

## Submission Notes
- Keep changes minimal and reproducible
- Include command(s) used for validation
- Prefer additive docs/examples over large refactors
