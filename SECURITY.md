# Security Policy

## Reporting Security Issues

If you discover a security vulnerability, unsafe behavior, license abuse, or misuse of this SDK, please report it privately.

Contact: ice____@msn.com

Please include, when possible:

- A clear description of the issue
- Steps to reproduce
- Affected files, modules, or versions
- Potential impact
- Any suggested mitigation

## Responsible Disclosure

Please do not publicly disclose security issues before RetryIX has had a reasonable opportunity to review and respond.

We appreciate responsible reports from researchers, developers, and users.

## Scope

This security policy applies to the public Möbius Phase Retrieval SDK repository.

It does not grant access to, or rights over, private RetryIX runtime internals, MemoryKernel components, VirtualPIM execution paths, TurboQuant-compatible backend internals, SPD / EEPROM physical interface designs, or other patent-sensitive RetryIX technologies.
